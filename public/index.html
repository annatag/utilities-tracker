House Maintenance, Utilities, Contacts & Finance Tracker

Tracks maintenance, utility bills, service contacts, mortgages, insurance, property taxes, sales tax, and tourist tax. Data syncs to Cloudflare KV when cloud storage is connected and falls back to this browser when offline.
Checking shared cloud storage…
Do not store passwords here. Use your browser or password manager for credentials. This app stores provider links, account notes, bill totals, due dates, autopay, and PAID, NOT PAID, or INACTIVE status only.
Others: Use “+ Add bill” and choose Auto Insurance, Car Payment, Mobile Phones, Math Class, Soccer Club, or Custom bill type from the bill type dropdown.
Tip: in cloud mode, data is shared through Cloudflare KV. Auto-backup still downloads a JSON safety copy when data changes.
